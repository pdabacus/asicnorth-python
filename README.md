 * [tesseract](https://github.com/tesseract-ocr/tesseract)
 * [python 3.7](https://www.python.org/downloads/)

   * jupyterlab
   * numpy
   * scipy
   * matplotlib
   * pandas
   * pandas-datareader
   * pillow
   * pytesseract
   * tensorflow
   * keras

download the notebooks in this repo and view everything with `jupyter lab /path/to/asicnorth-python/asicnorth/`
